# SafeEnum

## Installation


**`build.gradle`**
``` groovy
plugins {
  id "com.github.smac89.aspectj" version "0.1.2" // Or any plugin for aspects
}

repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    compileOnly 'com.github.smac89:safeenum:v0.3.0'
}
```
